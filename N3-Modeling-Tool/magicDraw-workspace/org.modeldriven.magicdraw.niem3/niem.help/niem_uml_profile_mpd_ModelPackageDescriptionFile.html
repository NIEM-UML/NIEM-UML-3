<<ModelPackageDescriptionFile>>

The ModelPackageDescriptionFile stereotype applies to a Usage dependency that represents a relationship between an MPD or a file set and an artifact (generally a namespace) to be included in the MPD. Reference Sections 4.2.3 and 4.2.4 of NIEM MPD Specification v1.0 (http://reference.niem.gov/niem/specification/model-package-description/1.0/).
