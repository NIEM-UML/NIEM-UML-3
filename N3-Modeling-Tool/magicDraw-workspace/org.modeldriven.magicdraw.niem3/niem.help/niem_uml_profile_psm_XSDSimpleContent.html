<<XSDSimpleContent>>

XSDSimpleContent represents a relationship between two type definitions: the first is a complex type definition with simple content whose content type is the second. This relationship is implemented in XML Schema through base attribute on the xsd:extension or xsd:restriction element of the first type definition, the actual value of which resolves to the second type definition. Section 3.4 of XML Schema Structures addresses simple content types in XML Schema; Sections 6.5.1, 6.5.2, and 7.4 of NIEM NDR v1.3 address simple content types in NIEM-conformant XML Schema.
